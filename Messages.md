
## JAVA to Arduino

  ISS - Initialize Sporadic Server
  
  APT - Add Aperiodic task
  
  PET - Add Periodic task
  
  SPT - Stop Periodic task
  
  ABT - Add Batch of tasks
  
  MSC - Get Servers max capacity  

## Arduino to JAVA
  
  SYS - Sistems status
  
  TMA - Tasks markers
  
  MSG - Arduino String Message
